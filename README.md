# Library Management System (Flask)

A **role-based Library Management System** built using **Python Flask**, where an **Admin (Librarian)** can manage books and a **User (Student)** can view, issue, and return books.

---

## Features

- **Role Selection**: Admin or User
- **Admin Functionality**:
  - Add new books with title, author, and number of copies
  - View all existing books with available copies
- **User Functionality**:
  - View all books in the library
  - Issue a book (reduces available copies)
  - Return a book (increases available copies)
- **Automatic inventory tracking** (count of available books)
- Simple and clean **UI using HTML & CSS**

---

## Tech Stack

- **Backend**: Python, Flask  
- **Frontend**: HTML, CSS  
- **Version Control**: GitHub
